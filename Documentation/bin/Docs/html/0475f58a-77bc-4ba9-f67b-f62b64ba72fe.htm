PluggedMethodAttribute Class

Indicates a method is plugged.

Use the ASMFilePath without any file extensions to specify the path to the ASM file that contains the plug. The extension is automatically added based on the target architecture. E.g. "PlugFileName.x86_32.asm". See remarks for more info.

Remarks

This attribute must be applied to a method which is plugged to tell the compiler where the ASM for the plug can be found.

The file name should be a relative path, relative to the build directory. The file name extension is added automatically so different ASM plugs can be used for different target architectures.
