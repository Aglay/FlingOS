ILCompiler Methods

The ILCompiler type exposes the following members.

Methods
  Name Description
Cleanup
Cleans up any temporary files, directories or streams.
ExecuteASMSequencer
Initialises the ASMSequencer then executes the ASMSequencer. It is assumed the ILScanner has been executed successfully. It will then output all the ASM chunks from the ASMSequencer to file.
ExecuteILReader
Initialises the ILReader then executes the ILReader
ExecuteILScanner
Initialises the ILScanner then executes the ILScanner. It is assumed the ILReader has been executed successfully.
ExecuteNASM
Executes NASM on the output file. It is assumed the output file now exists.
ExecuteProcess
Uses Process class to start a new instance of the specified process on the machine with specified start arguments. Note: This is a blocking function. Note: Waits a maximum of 15 minutes before assuming the process has failed to execute.
Finalise
Finalises the compilation by turning the .BIn file into a usable .ISO file.
InitASMSequencer
Intialises TheASMSequencer.
InitEnvironment
Initialises the compiler environment including initialising TheAssemblyManager.
InitILReader
Intialises TheILReader.
InitILScanner
Intialises TheILScanner.
