Aims and Ideas

While clearly an operating system has main tasks to fulfill, especially in the modern era, it is important to have a set basic aims and ideas to work towards. Therefore, this content is grouped into two sections: Aims..and Ideas.

Aims

Aims are ideals that Fling OS should try to achieve.

  • Kernel Modularisation

  • Well-documented code

Ideas

Ideas are well..ideas. They the inspiration for the design of Fling OS and what sets it apart from existing operating systems.

  • App-hardware independence

  • Flinging

  • Low-level managed memory

  • Remoting

  • Total memory virtualisation
