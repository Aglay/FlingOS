Kernel.Compiler Namespaces
Contains namespaces related to the kernel compiler.
Namespaces
Namespace Description
Kernel.Compiler
Contains the code for compiling kernel IL code into the final assembler file.
Kernel.Compiler.App
The basic compiler application that uses Kernel.Compiler.
Kernel.Compiler.Architectures.x86_32
Contains the Il Op conversion class implementations for converting IL op to valid x86 - 32 bit assembler.
Kernel.Compiler.ILOps
Contains all the IL to ASM conversion classes for supported IL ops.
