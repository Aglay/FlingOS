FATFileStream Methods

The FATFileStream type exposes the following members.

Methods
  Name Description
GetActualSize
Gets the actual length (size) of the stream. For files, this is FileSize. For directories, it is calculated from the number of clusters times cluster size.
GetClusterNums
Gets the list of cluster numbers that are part of the file being read/written from/to.
Read
Reads the specified number of bytes from the stream from the current position into the buffer at the specified offset or as many bytes as are available before the end of the stream is met.
(Overrides StreamRead(Byte, Int32, Int32).)
Write
Writes the specified number of the bytes from the buffer starting at offset in the buffer.
(Overrides StreamWrite(Byte, Int32, Int32).)
