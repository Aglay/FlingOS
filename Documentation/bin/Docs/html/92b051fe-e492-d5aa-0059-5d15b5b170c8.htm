ILOpInfo Fields

The ILOpInfo type exposes the following members.

Fields
  Name Description
ASMEndPos
The ASM (text) index where the ASM for this ILOp ends.
ASMStartPos
The ASM (text) index where the ASM for this ILOp starts.
DBILOpInfo
The debug database record asscoiated with this IL op.
IsDebugOp
Indicates whether the IL Op is a debug op.
LoadAtILOffset
The offset (in IL bytes) to load the method token at.
MethodToCall
When the IL op is a Call, this field holds the value of the method base to call. It should be used in preference to getting the method base from the metadata token in the ValueBytes. When the IL op is a Ldftn, this field holds the value of the method base to load. It should be used in preference to getting the method base from the metadata token in the ValueBytes.
NextPosition
The position of the next IL op in the same method. (-1 if none)
opCode
The OpCode (from System.Reflection.Emit) for this IL op. Note: opCode.Value == IlOps.ILOp.OpCodes - they can be used interchangeably.
Position
The position of this IL op in its method.
SetToGCDecRefCountMethod
Whether to set the MethodToCall to the kernel's GC Decrement Ref Count method.
ValueBytes
The value bytes loaded with this IL op (which, for example, are used to specify string values.)
