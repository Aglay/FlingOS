String Methods

The String type exposes the following members.

Methods
  Name Description
Concat
Concatenates two strings into one new string.
GetCharPointer
Gets a pointer to the first character in the string.
GetCharPointer(String)
Gets a pointer to the first char (that represents a character) of the specified string.
GetLength
Gets the length of the specified string.
New
Creates a new, blank FOS_System.String of specified length. IMPORTANT NOTE: You MUST assign the return value of this to a variable / local / arg / field etc. You may not use IL or C# that results in an IL Pop op of the return value of this method as it will screw up the GC RefCount handling.
PadLeft
Creates a new string and pads the left side of the string with the specified character until the whole string is of the specified length.
PadRight
Creates a new string and pads the right side of the string with the specified character until the whole string is of the specified length.
Split
Splits the string at every index where splitChar occurs and adds the splits parts (excluding splitChar) to a list of strings.
StartsWith
Determines whether the string starts with the specified string.
Substring
Creates a new string that is a copy of the current string starting at the specified index for specified length.
ToUpper
Copies the current string then converts all the alpha-characters to upper-case.
Trim
Creates a new string and trims all spaces from the beginning and end of the string.
TrimEnd
Creates a new string and trims all spaces from the end of the string.
