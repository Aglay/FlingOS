BasicDebug Methods

The BasicDebug type exposes the following members.

Methods
  Name Description
BeginEnableDebug
Begins enabling the debug interrupt handler
EndEnableDebug
Ends enabling the debug interrupt handler
Execute
The main execute method for the basic debugger
Init
Initialises the basic debugger
InitSerial
Initialises COM1 as serial connection to debug over
InsertCommandsList
Inserts the debug commands into the ASM.
Int3
Invokes interrupt 3.
InterruptHandler
Inserts the plug for the Int1 and Int3 interrupt handler.
SendArguments
Sends the arguments values as they were before the interrupt. Requires the debugger to send it how many bytes for the arguments there are.
SendBreakAddress
Sends the address of the last instruction that executed when the break occurred.
SendBreakCmd
Sends the Break command.
SendLocals
Sends the locals values as they were before the interrupt. Requires the debugger to send it how many bytes for the arguments there are.
SendRegisters
Sends the register values as they were before the interrupt.
Serial_ReadByte
Reads a byte from the debug serial port. Not callable from C#.
Serial_ReadUInt16
Reads a UInt16 from the debug serial port. Not callable from C#.
Serial_ReadUInt32
Reads a UInt32 from the debug serial port. Not callable from C#.
Serial_SafeReadUInt32
Safely reads a UInt32 from the debug serial port
Serial_WriteByte
Writes the specified value to the debug serial port. Not callable from C#.
Serial_WriteString
Writes the specified value to the debug serial port. Not callable from C#.
Serial_WriteUInt16
Writes the specified value to the debug serial port. Not callable from C#.
Serial_WriteUInt32
Writes the specified value to the debug serial port. Not callable from C#.
WaitForCommand
Waits for a command from the debugger.
