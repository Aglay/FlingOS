Kernel.FOS_System.IO Namespace
Classes for managing input/output tasks such as file handling.
Classes
  Class Description
Base
Represents any listing - a directory or a file.
Directory
Represents a directory (folder) listing.
File
Represents a file listing in a file system.
FileSystem
Represents a file system which must exist within a partition.
FileSystemManager
Provides management for file systems in the kernel.
FileSystemMapping
Represents a file system mapping. A file system mapping maps a path prefix (e.g. A:/) to a particular file system.
IOException
Represents an IO exception.
Partition
Represents a partition on a disk drive.
