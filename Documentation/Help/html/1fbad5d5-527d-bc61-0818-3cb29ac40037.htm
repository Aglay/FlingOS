Kernel.Compiler.ILOps Namespace
Contains all the IL to ASM conversion classes for supported IL ops.

StackSwitch
Custom IL op that rotates (upwards) the specified number of top-most stack items with eachother. Does not alter compiler record of stack - caller must do that. Acts totally ignorant of item types (e.g. int or float). Only moves dwords. Number of dwords to move should be in value bytes. Null value bytes indicates rotate (switch) top two items.
