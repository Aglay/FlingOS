Methods

Create
Creates a file stream for the specified file. Handles creating the correct type of file stream for the file system to which the file belongs.
Read
Reads the specified number of bytes from the stream from the current position into the buffer at the specified offset or as many bytes as are available before the end of the stream is met.
(Inherited from Stream.)
Write
Writes the specified number of the bytes from the buffer starting at offset in the buffer.
(Inherited from Stream.)
