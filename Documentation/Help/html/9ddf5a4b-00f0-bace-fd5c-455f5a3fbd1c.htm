AssemblyManager Fields

The AssemblyManager type exposes the following members.

Fields
  Name Description
Assemblies
The dictionary of all the assemblies loaded by the assembly manager. The key is the assembly’s full name. The value is the loaded assembly.
PluggedTypes
The list of all the types that contain plugs loaded by the assembly manager. The key is the type’s full name. The value is the loaded type.
UnpluggedTypes
The list of all the types that don’t contain plugs loaded by the assembly manager. The key is the type’s full name. The value is the loaded type.
